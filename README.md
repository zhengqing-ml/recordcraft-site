# RecordCraft Static Site

This is the static website repository for RecordCraft, primarily serving as the official home page and providing required App Store Connect URLs (Privacy Policy, Support, etc.).

## Directory Structure
- `index.html`: Home page
- `privacy.html`: Privacy Policy
- `terms.html`: Terms of Service
- `support.html`: Support
- `contact.html`: Contact Us
- `privacy-choices.html`: Privacy Choices
- `assets/`: CSS, JS, Images, and Icons

## How to deploy on GitHub Pages
1. Push this directory to a new GitHub repository (e.g., `recordcraft-site`).
2. Go to the repository **Settings** -> **Pages**.
3. Under **Build and deployment**, select **Deploy from a branch**.
4. Set the branch to `main` and folder to `/ (root)`.
5. Click **Save**. Keep an eye on the Actions tab to see the deployment progress.
6. Once deployed, get your URL from the Pages settings and use it for App Store Connect!
